# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-06

### Added
- Initial workflow kit structure (`agents`, `commands`, `rules`, `CLAUDE.md`).
- Safe installer with backup behavior.
- Public docs (`README.md`, `INSTALL.md`, `LICENSE`).
- Security policy (`SECURITY.md`).
- Uninstall helper (`uninstall.sh`).
- CI workflow for shell/json/yaml validation.
- Optional command-based help (`commands/help-workflow.md`).

### Changed
- Installation instructions now target repository root `install.sh`.
- Installer now validates JSON before merging settings.
